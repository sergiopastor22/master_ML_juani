# Ejercicio de estadística clásica: Chebyshev, Hoeffding y regla 68–95–99.7

Este repositorio contiene un notebook de Jupyter donde se estudian
numéricamente distintas distribuciones (normales, uniformes y betas) y se
comparan:

- los intervalos de confianza para la media construidos con las desigualdades
  de **Chebyshev** y **Hoeffding**,
- las estimaciones empíricas frente a los valores **teóricos** de cada
  distribución (media, varianza, ancho de los IC),
- el cumplimiento (o no) de la regla **68–95–99.7** en cada caso.

El trabajo está basado en las instrucciones de los tutoriales proporcionados en el repositorio, especialmente el tercero.

## Contenido del repositorio

- `Ejercicio_juani.ipynb`: notebook principal con el desarrollo completo de los
  tres problemas.
- `tut1_probability_optimization.ipynb`
  Noteboon de epaso de probabilidad, optimización y desigualdades.
  
- `tut2_classic_statistics_point_estimation.ipynb`  
  Notebook del curso sobre estimación puntual.

- `tut3_classic_statistics_confidence_intervals.ipynb`  
  Notebook del curso sobre intervalos de confianza.
